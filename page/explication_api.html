L'API utilisé est : spoonacular

C'est une API qui est repartie en 4 grandes parties :

  • Les ingrédients
  • Les recettes
  • Les produits
  • Les éléments de menu
Pour utiliser les différentes recherche il faut écrire en anglais par exemple apple,apple cake, pasta... et certaine recherche avec trop de mots où des mots compliqué ne fonctionne pas

Chaque partie contient différentes fonctionnalités recherchées par nom, récupérer des informations grâce à un id....

Attention il y a un maximun de requête par jour, normalement il est possible d'exécuter plusieurs fois les différentes requêtes,juste pas trop refresh la page gallery et tableau car ce sont les requêtes qui "consomment" le plus

Les différents endpoints utilisés sont :

  • https://api.spoonacular.com/recipes/findByNutrients?apiKey=f70a9841f245471db500d6789916d2b4&minCarbs=2&number=12 (tableau)
  • https://api.spoonacular.com/recipes/complexSearch?apiKey=f70a9841f245471db500d6789916d2b4&query=apple&number=20 (gallery image)
  • https://api.spoonacular.com/food/search?apiKey=f70a9841f245471db500d6789916d2b4&query='+encodeURI(rplat)+'&number=4 (recherche par aliment)
  • https://api.spoonacular.com/food/menuItems/search?apiKey=f70a9841f245471db500d6789916d2b4&query='+encodeURI(rplat)+'&number=12 (recherche plat restau)
  • https://api.spoonacular.com/recipes/complexSearch?apiKey=f70a9841f245471db500d6789916d2b4&query='+encodeURI(rplat)+'&number=12 (recherche de plat recette)

Le lien de la doc officiel : spoonacular

J'appelle l'API sur les pages :

  • Recherche plat (restaurant), pour chercher un plat et afficher les différents restaurants qui propose ce plat, la recherche se fait avec l'insertion d'un plat et l'action de la recherche avec le bouton avec la fonction javascript
  • Recherche de plats (recettes), pour chercher un plat, afficher une image du plat et un bouton pour pouvoir voir la recette,la recherche se fait avec l'insertion d'un plat et l'action de la recherche avec le bouton avec la fonction javascript
  • Recherche par aliments, pour rechercher les recettes,les produits et les vidéos en lien avec l'aliment rechercher, la recherche se fait avec l'insertion d'un aliment et l'action de la recherche avec le bouton avec la fonction javascript
  • Gallery image,pour pouvoir afficher les différentes image des recettes avec en paramètre l'aliment apple,la recherche se fait avec le chargement de la page html avec une fonction javascript
  • Tableau, pour pouvoir afficher les différentes caractéristiques d'un plat gramme de sucre, matière grasse,protéine et pouvoir cliquer sur la recette la recherche se fait avec le chargement de la page html avec une fonction javascript
